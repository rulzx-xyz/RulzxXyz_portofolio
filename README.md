### 🌑 Tentang Repositori
Repositori ini adalah pusat kode sumber (*source code*) untuk portofolio web profesional pribadi milik **Arul**. Mengusung estetika visual yang gelap, tajam, dan minimalis, situs ini dirancang untuk menampilkan arsip proyek, keahlian teknis, serta identitas visual secara interaktif.

### 🛠️ Tech Stack & Keahlian Utama
* **Frontend Development:** HTML5, CSS3 (Glassmorphism & Custom Animations), JavaScript (ES6+).
* **Desain Komunikasi Visual (DKV):** Adobe Photoshop, Illustrator, UI/UX Design (Figma), Typography & Visual Hierarchy.
* **Automation & Core Systems:** Node.js, WhatsApp Bot Integration (Baileys), API Integration (Claude AI).
* **Deployment & Hosting:** Vercel & Git Version Control.
